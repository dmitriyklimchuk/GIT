репозиторій - сховище(щось на зразок гуг драйва) - локальний (ваш компютер) 
, є віддалени (оріджин) той який в мережі інтернет
Git Hub, Bitbucket, Git Lab

бренч - branch - гілка версія репозиторія яка є відокрмленоє від нього (репозиторія) для додавання змін

індекс - index - стан при якому выдстежуються зміни файлів

коміт - commit - стан зберігання змін

push - зааплоадити зміни в репозиторій

pull - зтянути зміни з репозиторія(віддаленого)

мастер, девелоп - назви бренчів, за замовчуванням при ініціалізації

ініціалізація Гіта - створення гіт директорія

COMMANDS

git init - ініціалізаці ГІТа, у вашому фолдері, проєкті.

git help - помічник по ГІТу (команди)

git add filename - додати в індекс файл 

git add . додати ВСІ файли до індекса
